# Chatbot Universitario

Proyecto académico desarrollado como parte de la **Actividad 6** de la asignatura correspondiente, cuyo objetivo es la creación de un chatbot informativo para un entorno universitario utilizando tecnologías web básicas.

---

##  Descripción del proyecto

Este chatbot permite responder preguntas frecuentes mediante un archivo JSON, simulando una conversación básica con el usuario. Está diseñado para funcionar de manera local en el navegador sin necesidad de servidor ni base de datos.

El proyecto tiene fines educativos y demuestra el uso de JavaScript para el manejo de eventos, lectura de datos y presentación dinámica de información.

---

##  Tecnologías utilizadas

- HTML
- CSS
- JavaScript
- JSON
- Visual Studio Code
